## [1.0.0] - 2025-01-04

### Added
- Full-stack job board application
- FastAPI backend with async SQLAlchemy
- Next.js 14 frontend with App Router
- OpenAI GPT-4 powered AI assistant
- HeadHunter API integration for real job data
- JWT-based authentication system
- OAuth support
- Email verification
- Password reset functionality
- 33 E2E tests with Playwright (100% pass rate)
- HTTPS support for development and production
- Rate limiting
- CORS configuration
- Database migrations with Alembic
- Comprehensive API documentation (OpenAPI/Swagger)
- Responsive UI with Tailwind CSS and shadcn/ui
- AI chat widget with streaming responses
- Job search with filters (location, level, keywords)
- Company profiles
- Job bookmarking
- Real-time job sync from HeadHunter API
